Оператор сортировочной горки — специалист, отвечающий за «роспуск» вагонов. Расцепить для перераспределения по новым составам тысячи вагонов в сутки, чтобы каждый из них отправился вовремя в нужном направлении задача не из легких. С решением такой задачи ежедневно сталкивается оператор сортировочной горки, специалист, обладающий уникальными навыками.
